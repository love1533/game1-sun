# Game Orchestrator

한자 학습 게임 프로젝트의 개발을 총괄하는 오케스트레이터.

## 에이전트 구성

| 에이전트 | 역할 | 스킬 | 출력 |
|---------|------|------|------|
| game-developer | 한자 게임 개발 | game-dev | app/games/*/page.tsx |
| qa-agent | 품질 검증 | game-qa | 검증 보고서 |

## 워크플로우

### Phase 0: 컨텍스트 확인
1. 프로젝트 디렉토리에 `package.json` 존재 여부 확인
2. 실행 모드 결정:
   - 수정 요청 → 부분 재실행 (해당 게임만)
   - 새 게임 요청 → 추가 실행

### Phase 1: 게임 개발 (팬아웃)
game-developer 에이전트를 게임별로 병렬 실행:

1. **한자왕** (hanja) - 한자 급수별 퀴즈 게임 (8급~1급)

향후 추가 가능 게임:
- 한자 쓰기 연습
- 한자 매칭 게임
- 한자 사자성어 퀴즈
- 한자어 단어 조합

### Phase 2: QA 검증
qa-agent로 전체 프로젝트 검증

### Phase 3: 배포
1. Vercel CLI로 배포: `npx vercel --yes`
2. 배포 URL 사용자에게 전달
