# Game QA Skill

한자 학습 게임 프로젝트의 품질을 검증한다.

## 검증 단계

### 1. 빌드 검증
```bash
npm run build
```
빌드 실패 시 에러를 분석하고 직접 수정한다.

### 2. 구조 검증
- 모든 게임 라우트 존재 확인 (app/games/*/page.tsx)
- 'use client' 디렉티브 확인
- 메인 허브(app/page.tsx)에서 모든 게임 링크 확인

### 3. 한자 데이터 검증
- 한자 데이터 정확성 확인
- 급수별 분류 확인
- 음/훈 정확성 검증

### 4. 모바일 호환성
- viewport 설정 확인
- 터치 이벤트 확인
- 반응형 레이아웃 확인

## 발견된 문제 처리
- TypeScript 에러: 직접 수정
- 빌드 에러: 직접 수정
- 한자 데이터 오류: 보고서에 기술
